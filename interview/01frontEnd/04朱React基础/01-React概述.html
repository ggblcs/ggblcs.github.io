# 01 -React概述

# React 介绍

目标：了解什么是 React？

!---image-20211006124106131

Reactopen in new window 是一个专用于构建 用户界面 的JavaScript 库。

vue 框架 : 一个完整的解决方案(洗衣机)

react 库: 方法的集合(搓衣板)

它起源于 Facebook 公司的内部项目

  • 2011 年：对外透露 React 项目
  • 2012 年：对外公开 React 开发的应用 Instagram
  • 2013 年：正式开源

React 是目前所有前端 框架/库 中流行度最高的一个 （各框架流行度数据对比）open in new window

总结

React 是一个用于构建用户界面的 Javascript 库

# React 特点

目标：了解 React 的三大特点

  1. 声明式

  • 使用 “标记语言” 描述界面的样子

  • 数据驱动视图：当数据发生改变时，界面随之重新渲染

【编程语言小知识】

声明式语言：XML、HTML、JSON、... （描述结构的语言；功能简单）

命令式语言：JavaScript、Java、C、... （描述行为的语言；功能丰富）

<div className="app">
  <h1>Hello React! </h1>
  <h2>动态数据变化: {count}</h2>
</div>
  1. 组件化

组件化编程是现代前端框架的核心，也是 React 的 核心

  • 组件可以封装页面中的 部分或全部内容

  • 组件可以 组合、复用，形成完整的页面

!---image-20211006125027421

  1. 学习一次，跨平台编写. Taro. Uniapp

  • 可以开发 Web 应用：react-dom

  • 可以开发原生移动应用：react-native

  • 可以开发 VR 应用：react360

!---image-20211006125918723

总结

React 的三大特点是哪些？

React 对于我们的意义：

  • 求职工资高、去大厂的必备技能
  • 掌握 Vue 和 React 双框架，选择机会面更广
  • 对比 Vue 和 React 框架，能加深对前端框架的本质理解

# 运行项目

**目标：**掌握如何启动 react 项目

在新建项目 package.json中包含的 4 个命令：

"scripts": {
  "start": "react-scripts start",
  "build": "react-scripts build",
  "test": "react-scripts test",
  "eject": "react-scripts eject"

启动项目：

npm run start
yarn start

总结：

怎么启动项目，主要是看 package.json 提供了什么命令

# 用 React 编写网页内容

目标：掌握用 React 渲染页面内容的最基本写法

用原生 JS 来动态创建 HTML 页面元素的方式是怎么样的？

基本步骤：

  1. 导入依赖包：react和react-dom
  2. 创建要显示到页面中的 React 元素 (即虚拟 DOM 元素)
  3. 将 虚拟 DOM 元素 变成 真实 DOM 元素 ，渲染到页面中的某个位置

核心代码

  1. React 17 之前的写法：
// 导入依赖包
import React from 'react'
import ReactDOM from 'react-dom'

// 创建 React 元素
const el = React.createElement('h1', null, 'hello react')

// 渲染 React 元素
ReactDOM.render(el, document.getElementById('root'))
  1. React 18 的写法：
// 导入依赖包
import React from 'react'
import ReactDOM from 'react-dom/client'

// 创建 React 元素
const el = React.createElement('h1', null, 'hello react')

// 渲染 React 元素
ReactDOM.createRoot(document.getElementById('root')).render(el)

利用第二个参数添加属性

  • className
  • id

利用第三个参数添加子元素

  • 数组(少见)
  • 也可以作为第四个 第五个 ... 参数传递
  • 字符串

总结

  1. 脚手架工具生成的项目中，已默认安装了 react 和 react-dom ，无需额外安装
  2. 创建 React 元素： React.createElement()
  3. 渲染 React 元素： ReactDOM.render()
